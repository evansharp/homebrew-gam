homebrew-gam
-----------
A Homebrew formula for joylee's GAM script.

* `gam` -- GAM is a command line tool for Google Apps Administrators

----

## Installation

```
brew tap evansharp
brew install gam
```

## Update

```
brew uninstall gam 
brew untap evansharp
brew tap evansharp
brew install gam
```
