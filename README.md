homebrew-gam
-----------
A Homebrew formula for joylee's GAM script.

* `gam` -- GAM is a command line tool for Google Apps Administrators

----

## Installation

```
brew tap goodeggs/it
brew install gam
```

## Update

```
brew uninstall gam 
brew untap goodeggs/it
brew tap goodeggs/it
brew install gam
```
